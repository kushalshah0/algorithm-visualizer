# Sorting Algorithm Visualizer

An interactive web application that visualizes how sorting algorithms work step-by-step.

## Features

- **Visual step-by-step execution** - Watch each comparison, swap, and iteration
- **Multiple algorithms** - Bubble Sort, Quick Sort, Merge Sort
- **Playback controls** - Play, pause, step forward/backward, adjust speed
- **Customizable arrays** - Generate random arrays or set custom values
- **Dark mode** - Toggle between light and dark themes
- **Algorithm details** - View time complexity and stability information

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:5173 in your browser.

## Usage

1. Select a sorting algorithm from the dropdown
2. Choose array size (5-50 elements) or enter custom values
3. Use playback controls to visualize the sorting process
4. Toggle dark mode via the theme button

## Tech Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS 4
